TASK: Create a PDF Document in Adobe Acrobat
PLATFORM: Adobe Acrobat (Pro or Standard, desktop)

Step 1: Open Adobe Acrobat.
Step 2: On the Home screen, click "Create PDF" in the left panel, or go to File > Create > PDF From File.
Step 3: Choose the source of the PDF:
   - From a file: navigate to the document (Word, Excel, image, etc.) and click "Open". Acrobat will convert it automatically.
   - Blank page: go to File > Create > Blank Page.
Step 4: Once the document opens, review the rendered content on the main canvas.
Step 5: (Optional) Edit the content if you have Acrobat Pro:
   - Click "Edit PDF" in the right-hand Tools panel.
   - Click on any text block to edit it directly.
   - Use the "Add Text" tool to insert new text boxes.
Step 6: (Optional) Add pages:
   - Go to Page Thumbnails (left sidebar), right-click a thumbnail, and choose "Insert Pages".
Step 7: (Optional) Add a header, footer, or page numbers:
   - Go to Tools > Edit PDF > Header & Footer > Add.
Step 8: (Optional) Add form fields (Acrobat Pro):
   - Go to Tools > Prepare Form. Acrobat auto-detects fields; add or adjust as needed.
Step 9: (Optional) Protect the document:
   - Go to Tools > Protect > Encrypt with Password.
   - Set a password for opening and/or editing.
Step 10: Save the file:
   - File > Save As > choose a location and filename.
   - Ensure the format is "Adobe PDF Files (*.pdf)".
Step 11: (Optional) Reduce file size:
   - File > Save As Other > Reduced Size PDF — useful for email attachments.

NOTES:
- Acrobat Reader (free) can only view PDFs; creating and editing requires Acrobat Standard or Pro.
- For a quick free alternative, use the online Adobe Acrobat web tool at acrobat.adobe.com.
