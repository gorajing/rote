TASK: Create a Photo Collage in Adobe Photoshop
PLATFORM: Adobe Photoshop (desktop)

Step 1: Open Adobe Photoshop.
Step 2: Create a new canvas: File > New.
   - Set width and height (e.g., 3600 x 2400 px for a landscape print at 300 dpi).
   - Resolution: 300 pixels/inch for print; 72 px/inch for web.
   - Color Mode: RGB.
   - Background: White.
   - Click "Create".
Step 3: Plan your layout. Decide how many photos to include and sketch a rough grid (e.g., 2 rows × 3 columns = 6 photos).
Step 4: Import the first photo: File > Place Embedded, navigate to the image file, and click "Place".
   - The image appears with transform handles. Resize and reposition it by dragging, then press Enter to confirm.
Step 5: Repeat Step 4 for each additional photo, placing them in their intended grid positions.
Step 6: Align and distribute photos evenly:
   - Select all photo layers (click first, then Shift-click the rest in the Layers panel).
   - Use Edit > Free Transform on each, or use the Align and Distribute options in the Options bar when multiple layers are selected.
Step 7: (Optional) Add a border between photos:
   - Select the Background layer.
   - Fill it with the desired border colour: Edit > Fill > Color.
   - Leave small gaps between photos so the background shows as a border.
Step 8: (Optional) Add a text label or title:
   - Select the Type tool (T key).
   - Click on the canvas and type your title.
   - Choose font, size, and colour from the Options bar.
Step 9: (Optional) Add a drop shadow to each photo for depth:
   - Select a photo layer, go to Layer > Layer Style > Drop Shadow, adjust settings, and click OK.
Step 10: Flatten and export:
   - Review the full composition.
   - File > Export > Export As. Choose format:
     - JPEG (quality 80–90 %) for sharing online.
     - PNG for a lossless version.
     - PDF for print.
   - Click "Export".
Step 11: Save the original layered file for future edits: File > Save As > Photoshop (.psd).

NOTES:
- Use Smart Objects (Place Embedded) rather than rasterising images so you can resize without quality loss.
- Group related layers (Ctrl/Cmd + G) to keep the Layers panel organised.
